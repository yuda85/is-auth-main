## Installation

Please run:

```bash
  npm install
  npm start
```

## Configuration

Change API URL in: `src/environments`

Change specific routes in: `src/app/services/auth.service.ts`

## Build & Deploy

Please run:

```bash
  npm run build
```

navigate to

```
path_to_app/dist
```

The built app should be there and after configuration it can be deployed to any server
